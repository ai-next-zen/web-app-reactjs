# ExecuCare Modern Website - Design Plan

## Design Direction

**Modern Professional Service Website** - A clean, trustworthy design that emphasizes professionalism and reliability for a carpet cleaning service company.

## Color Palette

The design will follow the original ExecuCare brand colors with modern refinements:

- **Primary Blue**: Deep navy blue (#1e3a8a to #1e40af) - conveys trust and professionalism
- **Accent Red**: Bold red (#dc2626 to #ef4444) - creates urgency and draws attention to CTAs
- **Neutral Grays**: Light backgrounds (#f8fafc, #f1f5f9) with dark text (#0f172a, #334155)
- **White**: Clean white (#ffffff) for cards and contrast

## Typography

- **Headings**: Inter or Poppins (bold, modern, professional)
- **Body Text**: Inter or Open Sans (clean, highly readable)
- **Accent**: Use font-weight variations for hierarchy

## Layout Approach

**Asymmetric Modern Layout** with the following principles:

1. **Hero Section**: Full-width background image with diagonal overlay, breadcrumb navigation, and centered heading
2. **Contact Cards**: 4-column grid on desktop (2x2 on tablet, stacked on mobile) with alternating blue/red backgrounds
3. **Two-Column Content**: Service areas list on left, contact form on right (stacked on mobile)
4. **Map Section**: Full-width interactive map with circular focus indicator
5. **Footer**: Dark navy background with organized columns

## Visual Style

- **Modern Cards**: Elevated cards with subtle shadows, rounded corners (8-12px radius)
- **Micro-interactions**: Hover effects on buttons and cards (scale, color transitions)
- **Icons**: Use lucide-react icons for contact info, locations, and form fields
- **Spacing**: Generous whitespace, consistent padding (16px, 24px, 32px, 48px scale)
- **Shadows**: Soft shadows for depth (shadow-md, shadow-lg)
- **Transitions**: Smooth 200-300ms transitions on interactive elements

## Key Features

1. **Responsive Navigation**: Sticky header with mobile hamburger menu
2. **Promotional Banner**: Eye-catching banner with special offers (dismissible)
3. **Contact Form**: Modern form with floating labels or clear placeholders
4. **Map Integration**: Google Maps with custom styling to match brand colors
5. **CTA Buttons**: Bold, high-contrast buttons with clear hierarchy
6. **Service Areas**: Organized list with location pins, easy to scan

## Accessibility

- High contrast ratios for text (WCAG AA minimum)
- Focus indicators on all interactive elements
- Semantic HTML structure
- Alt text for all images
- Keyboard navigation support

## Performance

- Optimized images (WebP format where possible)
- Lazy loading for below-fold images
- Minimal JavaScript for core functionality
- Fast initial load time
